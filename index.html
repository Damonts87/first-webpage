User Experience and User Interface

UX: User Experience

User experience design is importnat to help deliver the most effective use of the prodect as possible. The focus is on the full experince from the first to last contact by a user. This creates structural design solutions for any issues that someone may encounter.

UI: User Interface

User interface design is important to help deliver a product that is aesthetically delightful to users. The focus is on the visual aspect of what users will be dealing with. This creates combinations of colors, imagery, animations, and typography to make something the users will like looking at.

Jobs that use UX and UI include:

  • Game Developer
  • Digital Marketer
  • Multimedia Programmer
  • Many more
